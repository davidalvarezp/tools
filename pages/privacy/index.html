Privacy

Local-First Tools is designed with privacy as a core principle. This website does not collect, store, or process any personal data.

Data collection

No personal data is collected. Files you use with the tools never leave your device and are never uploaded to any server.

  • No file uploads
  • No form submissions
  • No server-side processing

Tracking & analytics

This website does not use analytics, tracking scripts, cookies, or third-party monitoring services.

  • No cookies
  • No tracking pixels
  • No fingerprinting

Local processing

All tools run entirely in your browser using client-side JavaScript. Once the page is loaded, the tools can work offline.

Third-party services

The website does not rely on third-party services to process data. External links (such as GitHub) are provided for convenience only.

Changes

If this privacy policy changes in the future, updates will be reflected on this page.

Your data stays on your device. Always.
