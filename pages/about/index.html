About

Local-First Tools is a collection of small, focused web tools designed to run entirely in your browser.

The goal is simple: provide useful utilities without collecting data, tracking users, or uploading files to external servers.

Why this project exists

Many so-called “online tools” require you to upload your files to a server. Even when unnecessary, your data leaves your device.

Local-First Tools takes a different approach: if something can be done in the browser, it should be done in the browser.

  • Your files never leave your device
  • No accounts or sign-ups
  • No cookies, no analytics, no tracking
  • Works offline after the first load

Technology

All tools are built using plain HTML, CSS, and JavaScript. There are no frameworks, no backend services, and no external APIs involved in processing your data.

This keeps the tools lightweight, auditable, and easy to host anywhere.

Open source

The entire project is open source and available on GitHub. Contributions, ideas, and feedback are welcome.

Local-first is not a feature — it’s a principle.
